# Tesseract Academy - Government Gateway

This is the official website for Tesseract Academy's Government Gateway, an enterprise-grade public sector procurement portal. The site facilitates government tenders, compliance checks, and consortium partnerships.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
